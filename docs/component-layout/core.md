## 属性快捷键 

>  点击某个键, 按下后弹出对话框进行快速操作,按下enter完成

L - layout , 弹出div的布局控制, 快速加入类, 和基础的padding, margin, border, 宽高设置
T - tree , 树形结构管理
S - style, 样式编辑器弹出层
A - action, 控制数据注入管理和内部数据绑定管理
